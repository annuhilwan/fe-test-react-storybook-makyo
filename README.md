
# Getting Started with Create React App

## Available Scripts

In the project directory, you can run:

### `1. install`
#### NPM
```bash
npm install
```
#### Yarn
```bash
yarn install
```

### `2. Start`
#### NPM
```bash
npm start
```

#### Yarn
```bash
yarn start
```

### `3. Storybook`
#### NPM
```bash
npm run storybook
```
#### Yarn
```bash
yarn storybook
```
